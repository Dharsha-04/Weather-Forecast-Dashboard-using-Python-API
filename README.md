TASK1 - API INTEGRATION AND DATA VISUALIZATION

Project Title

    Weather Forecast Dashboard using Python & API

Objective

    This project is developed as part of the CODTECH Internship Task to demonstrate API integration and data visualization using Python.

API Used:

    Data is fetched from the OpenWeatherMap API to retrieve real-time and 5-day weather forecast data.

Technologies Used

        Python

        Requests

        Pandas

        Matplotlib

        Seaborn

        Streamlit

        Geocoder

Features:
        ✔ Auto-detect user location 
        ✔ Manual city search 
        ✔ 5-day weather forecast 
        ✔ Temperature trend visualization 
        ✔ Weather condition distribution (Pie chart) 
        ✔ Data export to CSV 
        ✔ Interactive dashboard using Streamlit

Output Visualizations:

    Line Chart → Temperature Trend

    Pie Chart → Weather Condition Distribution

How to Run the Project:

    1.Install dependencies:

        pip install streamlit requests pandas matplotlib seaborn geocoder

    2.Add your API key in app.py

    3.Run the app:

        streamlit run app.py
